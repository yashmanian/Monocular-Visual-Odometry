Detailed Description

Definition of the epipolar class. It contains all the functions and variables required to compute epipolar geometry.

The sole purpose of this class is to compute geometry based on matched points. Inherits the PointOps class and uses the mathematical conversion methods from it.

Eigen::MatrixXd epipolar::computeEpipole ( Eigen::Matrix3d &  F)

Computes epipole based on the idea that the epipoles lie on the epipolar plane. So Linear Least Squares to solve e2T*F*e1 = 0.

Takes in the Fundamental matrix and returns a 3x2 matrix in homogenous form, with the coordinates for the epipole.

void epipolar::estimateFundamentalMatrix ( Eigen::MatrixXd &  points1,
Eigen::MatrixXd &  points2,
Eigen::Matrix3d &  F 
)

Estimates fundamental matrix from matched points. Takes in the matrix F by reference which is updated.

This version uses the 8 point algorithm and solves the Linear Least Squares problem Ax = 0 using SVD. The method can take more than 8 points and still compute the matrix. Takes in the matched points in the form of a 3xN matrix of homogenous coordinates.

Eigen::MatrixXd &  points2

Uses RANSAC to refine the Fundamental matrix. This takes advantage of the fact that the features lie on the epipolar plane.

Uses x2T*F*x1 = 0 as a distance function for the RANSAC. Once inliers are selected, a final fundamental matrix is computed using them.
